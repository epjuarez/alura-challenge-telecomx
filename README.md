# Alura Challenge Telecomx

Análisis de Evasión de Clientes

📌 Descripción del Desafío

Este proyecto corresponde a un desafío de análisis de datos enfocado en comprender y explicar la evasión de clientes (Churn) en un modelo de negocio basado en suscripción.

El objetivo principal fue realizar un análisis exploratorio completo del dataset para identificar patrones, variables relevantes y posibles factores asociados a la cancelación del servicio. El análisis culmina con la generación de insights estratégicos y recomendaciones orientadas a la reducción del churn.

La variable objetivo del estudio es:

* Churn
  - 0 = Cliente permanece
  - 1 = Cliente cancela
 
La tasa global de evasión identificada fue aproximadamente 26.5%, lo que representa un nivel significativo de pérdida de clientes y justifica un análisis detallado.

🎯 Objetivos del Proyecto

*  Explorar y comprender la distribución de la variable churn.
*  Analizar la relación entre churn y variables categóricas.
*  Evaluar el comportamiento de variables numéricas clave.
*  Identificar variables con mayor poder explicativo.
*  Generar conclusiones e insights accionables.
*  Proponer recomendaciones estratégicas basadas en datos.

🛠 Tecnologías Utilizadas

*  Python 3
*  Pandas = Manipulación y análisis de datos
*  Matplotlib = Visualización base
*  Seaborn = Visualización estadística avanzada
*  Google Colab = Entorno de análisis interactivo
